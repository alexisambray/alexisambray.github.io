Laboratory Lockdown Game

Lower-Order Game

Laboratory Lockdown is a 3D Unity game designed to help Grade 6 students in the Philippines master both lower- and higher-order learning outcomes on mixtures through interactive gameplay and real-world simulations.

The lower-order outcome focuses on fixture mixture classification such as true solution, solution, and suspension. The higher-order outcome challenges students to demonstrate various separation techniques, such as magnetic, scooping, filtration, decantation, picking, winnowing, sieving, and distillation.

Analyzing the Netflix Movies & TV Shows Dataset

Netflix Dataset Overview

Data Variables

Exploratory Data Analysis

This project is a case study on the Netflix Movies and TV Shows dataset from Kaggle, completed for our CSMODEL (Modelling and Simulation) course. Our goal was to analyze trends in content publishing and build a content-based recommender system using machine learning.

We began with data cleaning to ensure consitency and accuracy, followed by answering 9 exploratory questions (2 snippets shown above) using EDA techniques and Plotly visualizations. Our core research question was: "Would there be a significant difference in the performance of a recommender system based on structured metadata (title, director, cast, rating, genre) versus one based on textual descriptions?"

We built two recommendation algorithms, one based on metadata and one on descriptions. A Paired T-Test was performed to compare their accuracy, with results showing a statistically significant difference favoring one model. We concluded that description-based models yielded higher accuracy (t-value = 9.25, p = 1.51e-05).

This project was done by Loben Tipan, Kaitlyn Tighe, and myself.

  • Tools: Python, Pandas, Plotly, Seaborn, Jupyter Notebook
  • Key Topics: EDA, Data Cleaning, Recommender Systems, Hypothesis Testing, Cosine Similarity
  • Links: Kaggle | Notebook
