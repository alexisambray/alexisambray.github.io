02

Analyzing the Netflix Movies & TV Shows Dataset

EDA, recommender system, and statistical testing

A data analysis and modelling project using the Netflix Movies and TV Shows dataset from Kaggle. We cleaned the data, answered 9 EDA questions, built content-based recommender systems, and compared model performance using a paired t-test. Through exploratory analysis and machine learning techniques, the project revealed key content trends and strengthened data-driven decision-making skills.

  • Tools: Python, Pandas, Plotly, Seaborn, Jupyter Notebook
  • Topics: EDA, recommender systems, cosine similarity, hypothesis testing
  • Links: Kaggle | Notebook

04

StudioTrack

Wellness studio package and attendance tracker

StudioTrack is a mobile app case study designed for boutique fitness enthusiasts who attend classes across multiple studios and memberships. The app helps users track package balances, attendance streaks, class history, expiration dates, and cost per class in one organized place.

Problem: Boutique fitness users often manage multiple studio packages and struggle to track remaining classes, expiration dates, and spending across different platforms.

Solution: StudioTrack centralizes package management, attendance tracking, and fitness insights into one streamlined mobile app. Users can easily log classes, monitor remaining sessions, track package expirations, and understand their spending across multiple studios in a single organized dashboard.

  • Tools: Figma
  • Role: UI/UX Designer
  • Focus: Personal UX/UI Case Study, User Journey, Wellness Tracking, Product Thinking
  • Link: Figma Prototype
