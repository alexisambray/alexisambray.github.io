02

Analyzing the Netflix Movies & TV Shows Dataset

EDA, recommender system, and statistical testing

A data analysis and modelling project using the Netflix Movies and TV Shows dataset from Kaggle. We cleaned the data, answered 9 EDA questions, built content-based recommender systems, and compared model performance using a paired t-test. Through exploratory analysis and machine learning techniques, the project revealed key content trends and strengthened data-driven decision-making skills.

  • Tools: Python, Pandas, Plotly, Seaborn, Jupyter Notebook
  • Topics: EDA, recommender systems, cosine similarity, hypothesis testing
  • Links: Kaggle | Notebook
